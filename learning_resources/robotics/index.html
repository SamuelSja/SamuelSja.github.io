This is here to give a record of how we did robotics stuffs.
During 2024-2025 school year we used robot C to program the brain. I would not recommend using Robot C. It had terrible documentation, so it was a nightmare to program with. Durring 2025-2026 we used The Vex V5 system. I used the vscode extention to interface with the brain. Note that due to this we ran into an issue with a software update. We needed to downlaod an experimental version of the brains software. Because we had the wrong version one of our motors was overheating to quickly (or at least the motor thought it did), so if you are using vscode make sure you download and install the correct version of software. I do not think you can do this through vscode, so you will have to go to the Vex website. Although the vscode extention does have some setbacks, I found it really easy to use, so if you have used vscode and python before it is pretty cool to use. If you want the easy (and probably better) way you should just use the website. help avoid major issues such as not being able to find the updates.
